如你在使用《洪易奇门》过程中遇到问题，建议先查看下方“自助排查/常见问题”。仍无法解决时请通过邮件联系我们。

## 联系方式
- 技术支持邮箱：0927hzh@gmail.com
- 邮件标题建议：洪易奇门 技术支持｜问题简述

为提高定位效率，请在邮件中附上：
- 设备型号（如 iPhone 15 Pro / iPad 等）
- 系统版本（iOS / iPadOS / macOS / visionOS 版本）
- App 版本号（App 内设置页或 App Store 显示的版本）
- 问题发生时间与复现步骤（越具体越好）
- 截图/录屏（如有）
- 若涉及 AI 接口：可提供“服务商名称 + Base URL”，不要提供你的 API Key

## 自助排查（建议按顺序）
1. 退出 App 后重新打开
2. 确认系统时间与时区正确
3. 若涉及联网功能（AI、订阅恢复、iCloud 同步等）：检查网络后重试
4. 若涉及定位/真太阳时：到系统设置确认定位权限为“使用期间”并允许访问
5. 若是订阅解锁问题：先尝试“恢复购买（Restore Purchases）”

## 常见问题（FAQ）

### 订阅/会员相关
- 已购买 Pro 但未解锁？
  - 先尝试：恢复购买（Restore Purchases）
  - 确认购买所用 Apple ID 与当前设备登录 Apple ID 一致
  - 若仍异常，请邮件联系我们并说明：购买时间、是否更换过 Apple ID、是否多设备使用

- 恢复购买提示未发现订阅？
  - 可能原因：Apple ID 不一致 / 订阅已过期 / 网络异常
  - 建议：更换网络重试，并确认 Apple ID

### iCloud 同步相关
- 历史记录/设置没有同步？
  - 确认系统 iCloud 已登录、iCloud Drive 已开启
  - 等待片刻后重启 App
  - 多设备同步时，确保两台设备都开启 iCloud 同步

### 真太阳时/定位相关
- 无法获取位置或真太阳时不正确？
  - 到系统设置开启定位权限（使用期间）
  - 室内/无信号环境可能导致经纬度不稳定

### AI 功能相关（如启用了自定义服务商）
- 无法回答/一直报错？
  - 确认 Base URL 是否正确（不同服务商的路径可能不同）
  - 确认 API Key 已填写且仍有效（请勿在任何地方公开）
  - 尝试切换网络或更换服务商进行对比测试
